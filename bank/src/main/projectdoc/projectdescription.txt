Проект представляет собой RESTful API для управления списком задач.
Пользователи могут создавать, редактировать и удалять свои задачи.
Просмотр списка задач доступен всем пользователям.
Редактирование и удаление задачи может осуществлять только ее автор.
Для администратора предоставлены дополнительные права по управлению списком пользователей и списком задач.
